# FINAZACTIVOS Frontend

Portal de solicitud y evaluación de créditos para FINAZACTIVOS.

## 🚀 Quick Start

\`\`\`bash
npm install
cp .env.example .env.local
npm run dev
\`\`\`

Abrir [http://localhost:8000](http://localhost:8000)

## 📋 Features

- ✅ Formulario de solicitud multi-paso (6 pasos)
- ✅ Formulario de evaluación para asesores (6 tabs)
- ✅ Dashboard con filtros y búsqueda
- ✅ Autenticación con NextAuth
- ✅ Tests unitarios y E2E
- ✅ Accesibilidad WCAG 2.1 AA

## 🛠 Tech Stack

- Next.js 16 + React 19 + TypeScript 6
- Tailwind CSS 4
- React Hook Form + Zod
- Jest + Playwright
- NextAuth 5

## 📦 Scripts

\`\`\`bash
npm run dev          # Development server
npm run build        # Production build
npm test             # Unit tests
npm run test:e2e     # E2E tests
\`\`\`
